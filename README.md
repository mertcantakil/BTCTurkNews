
# BtcTurk PRO




## For Local Run
Clone the Project
```bash
  git clone https://github.com/mertcantakil/BTCTurk.git
```
Navigate to the Project Directory
```bash
  cd BTCTurk
```
Install the Required Packages
```bash
  npm install
```
Start the Server
```bash
  npm start
```

  
## Tech Stack
React, Typescript, Redux, CSS, Firebase, Ant-Design
